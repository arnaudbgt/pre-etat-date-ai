# Contexte produit - Pre Etat Date AI

## Objectif

Créer une plateforme permettant à un vendeur particulier de générer un pré-état daté à partir de documents de copropriété transmis en ligne.

Le service ne doit pas être présenté comme un document certifié par le syndic. Il doit être présenté comme un pré-état daté automatisé, généré à partir des pièces fournies par le vendeur, avec identification des champs manquants ou incertains.

## Client cible V1

Le client prioritaire est le vendeur particulier d’un lot en copropriété.

Son problème :

- le syndic facture souvent un montant élevé pour produire le document ;
- le vendeur ne sait pas quels documents fournir ;
- le notaire ou l’agent immobilier demande rapidement les pièces pour le compromis ;
- les données existent déjà dans les documents de copropriété.

## Promesse produit V1

Générer un pré-état daté complet ou quasi complet en quelques minutes, à partir des documents transmis par le vendeur.

Prix cible : 29 à 30 euros.

## Positionnement

Le produit doit être positionné comme un assistant intelligent de génération documentaire, avec :

- analyse automatique des documents ;
- demande guidée des pièces manquantes ;
- remplissage automatique du formulaire ;
- signalement des champs incertains ;
- génération PDF ;
- possibilité de contrôle humain ultérieur si nécessaire.

## Principe de prudence juridique

Ne pas promettre :

- un document certifié syndic ;
- une garantie absolue d’exactitude ;
- une substitution à l’état daté officiel.

Promettre :

- un document préparatoire ;
- un remplissage automatique à partir des documents fournis ;
- une indication transparente des sources et incertitudes.
