PROJECT MEMORY - PRE ETAT DATE

Vision

Pré Etat Date est un assistant spécialisé dans l’analyse documentaire de copropriété permettant à un vendeur particulier de générer un pré-état daté à partir de documents déjà en sa possession.

Le produit n’est pas présenté comme un simple générateur IA mais comme un moteur spécialisé copropriété.

⸻

Positionnement

Ne jamais communiquer :

* “Propulsé par ChatGPT”
* “Prompt ChatGPT”

Toujours communiquer :

* Analyse spécialisée des documents de copropriété
* Contrôle de cohérence
* Détection des pièces manquantes
* Score de confiance
* Validation utilisateur

⸻

Proposition de valeur

Le client ne paie pas pour un PDF.

Le client paie pour :

* l’analyse des documents ;
* la détection des incohérences ;
* la structuration automatique ;
* la génération d’un document exploitable.

⸻

Concurrents

Concurrent principal :

mon-pre-etat-date.fr

Forces observées :

* prévisualisation filigranée ;
* FAQ ;
* CGV simples ;
* tunnel de conversion efficace.

Faiblesses observées :

* peu de mise en avant de la fiabilité ;
* peu d’explication des sources ;
* absence de score de confiance ;
* positionnement très centré sur l’IA.
