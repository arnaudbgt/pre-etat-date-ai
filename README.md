# Pre Etat Date AI

Socle technique du MVP Pre Etat Date.

## Prérequis

- Node.js 20 ou supérieur
- npm 10 ou supérieur

## Démarrage

```bash
npm install
npm run dev
```

L'application est ensuite disponible sur `http://localhost:3000`.

## Vérifications

```bash
npm run format:check
npm run lint
npm run typecheck
npm run build
```
